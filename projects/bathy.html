Stony coral bed @ Gray's Reef. We used CARIS HIPS/SIPS and Base Editor to process the raw multibeam and sidescan sonar data; we then cross-referenced our processed imagery against previously collected data to see how the reef's morphology changed over time.
Prince of Wales Island. Multibeam data courtesy of NGDC; nautical chart courtesy of OCS. End product created entirely in CARIS HIPS/SIPS.
Asymmetric sand ripples at shallow depth. Data collected using a Kongsberg M3 and processed using Hypack and Fledermaus. Python scripts were used to fix ellipsoidal height issues resulting from a problematic tide file.
This was an experiment recreating the visual style of EarthRef Seamount Catalog. I used a combination of Python scripting (mostly using GDAL), MB-Systems (for building contours), and Photoshop.
